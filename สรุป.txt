For / While
 repetition structure
  เป็นชุดโปรแกรมที่ซ้ำกันรวมกันอยู่ ปัญหาที่ตามมา
   - ทำให้โปรแกรมมีขนาดใหญ่เกินไป
   - ใช้เวลาเยอะในการเขียน

   loop จะทำให้ชุดโปรแกรมที่ซ้ำกัน สั้นลง การทำงานคือ เช็คเงื่อนไข ถ้าเป็นจริง จะทำงานตามชุดโค้ดที่อยู่ภายใต้ loop

for loop ใช้สำหรับการวนลูปผ่านไอเทมต่าง ๆ ที่อยู่ในลำดับ เช่น รายการ (list), สตริง (string), หรือช่วงของตัวเลข (range)
 Syntax
  for variable in iterable:
    # บล็อกของโค้ดที่ต้องการให้ทำงานซ้ำ
    variable: ตัวแปรที่จะเก็บค่าของแต่ละไอเทมใน iterable ในแต่ละครั้งที่วนลูป
    iterable: คอลเล็กชันที่ต้องการวนลูป เช่น list, tuple, string, หรือ range
  ตัวอย่าง
   numbers = [1, 2, 3, 4]
   for num in numbers:
    print(num)

while loop จะทำงานซ้ำ ๆ จนกว่าค่าเงื่อนไขจะเป็นเท็จ (False)
 Syntax
  while condition:
    # บล็อกของโค้ดที่ต้องการให้ทำงานซ้ำ
    condition: เงื่อนไขที่จะตรวจสอบ ถ้าเงื่อนไขเป็นจริง (True) ลูปจะทำงาน ถ้าเป็นเท็จ (False) ลูปจะหยุด
  ตัวอย่าง
  count = 0
    while count < 5:
        print(count)
        count += 1

สรุป
for loop ใช้เมื่อคุณทราบจำนวนรอบที่จะวนลูป หรือมีคอลเล็กชันของไอเทมที่ต้องการวนลูปผ่าน
while loop ใช้เมื่อคุณไม่แน่ใจว่าจะวนลูปกี่รอบ แต่รู้ว่าต้องการให้ทำงานจนกว่าเงื่อนไขหนึ่งจะเป็นเท็จ
